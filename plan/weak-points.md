# Weak Points Tracker

If you are reading this after a gap, you are already succeeding.
The anxious person avoids this file. The serious person opens it.

---

Format (study gaps):  [date] | [topic] | [question failed] | [what was missed]
Format (drill gaps):  [date] | DRILL | [drill type] | [what broke] | [what you learned]

Review every Sunday. Same topic 3+ times = dedicated 30-min session needed.

Weekly questions to answer every Sunday:
  1. Which topic was hardest this week?
  2. Which concept felt most natural?
  3. One thing I can explain now that I could not last week.
